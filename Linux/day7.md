## Basic Concepts: Modes in Vim
Vim works in different **modes**, each designed for a specific task.

### Modes in Vim Editor
**vi / vim → Advanced and lightweight editor**

### Command Mode (Default Mode)
- Key: `Esc`
- Used for:
  - Shortcuts
  - Execution of commands
- Default mode when Vim starts

### Insert Mode
- Used for:
  - Writing data
  - Manipulating text
- Enter using:
  - `i`
- Exit using:
  - `Esc`

### Visual Mode
- Used for:
  - Selecting text
  - Copy, paste, delete operations
- Enter using:
  - `v`
- Exit using:
  - `Esc`

### Execute Mode (Ex Mode)
- Used for:
  - Saving files
  - Quitting Vim
- Enter using:
  - `:` (from command mode)
